# HackShield - Network Threat Analyzer & Password Security Auditor

HackShield is a full-stack cybersecurity dashboard built using Python (Flask) and Vanilla JavaScript. It helps analyze network security, detect vulnerabilities, and evaluate password strength with real-time insights.

## Features

* Multi-threaded TCP Port Scanner
* CVE-based Vulnerability Mapping & Risk Classification
* Password Strength Analyzer with Entropy Scoring
* Crack Time Estimation for Passwords
* HaveIBeenPwned API Integration for Breach Detection
* Real-time GeoIP Enrichment
* Scan History Tracking
* Downloadable Reports (.txt & .json)

## Tech Stack

* Backend: Python (Flask)
* Frontend: HTML, CSS, JavaScript
* Tools & APIs: Nmap, GeoIP, HaveIBeenPwned

## Installation & Setup

1. Clone the repository:
   git clone https://github.com/vipul87525/HackShield

2. Navigate to the project folder:
   cd HackShield

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   python app.py

5. Open in browser:
   http://127.0.0.1:5000/

## Usage

* Enter target IP/domain for network scan
* View open ports and detected vulnerabilities
* Check password strength and breach status
* Download reports for analysis
